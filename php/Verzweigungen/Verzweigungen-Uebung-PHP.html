PHP-Übung - Verzweigungen

Aufgabe 01

  • Schreiben Sie ein Formular zur Eingabe einer Jahreszahl.
  • Bei Klick auf Absendebutton wird eine zweite Seite aufgerufen, die die Eingabe auf Schaltjahr prüft und eine Ausgabe in der Form '2200 ist kein Schaltjahr' erzeugt.

Aufgabe 02

  • Schreiben Sie ein Formular zur Eingabe einer Monatszahl.
  • Bei Klick auf Absendebutton wird die die Anzahl der Tage in diesem Monat ermittelt und auf der gleichen Seite ausgegeben. Geben Sie für den Februar immer 28 Tage aus.

Aufgabe 03

  • Verpacken Sie die Ermittlung des Schaltjahres aus Aufgabe 01 in eine Funktion.
  • Benutzen Sie diese Funktion in der Lösung zu Aufgabe 02 um den Wert zum Februar korrekt anzupassen.
  • Verpacken Sie nun auch die Ermittlung der Monatstage in eine Funktion und verwenden Sie diese zur Ausgabe.
  • Im Formular muss jetzt natürlich Monat und Jahr eingegeben werden.
