PHP-Übung - Arrays

Aufgabe 01

  • Erzeugen Sie aus dem String "Montag, Dienstag, Mittwoch, Donnerstag, Freitag, Samstag" ein Array.
  • Geben Sie den Donnerstag aus.
  • Hängen Sie den Sonntag an.
  • Benennen Sie den Samstag in Sonnabend um.
  • Geben Sie das Array strukturiert aus.
  • Sortieren Sie die Wochentage alphabetisch absteigend.

Zusatz:

  • Ermitteln Sie den Index des Array-Elements Sonnabend (nicht abzählen!) und
  • benennen Sie es wieder zurück zu Samstag.
