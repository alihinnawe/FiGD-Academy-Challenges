PHP-Übung - Einbinden

Aufgabe 01

  • Schreiben Sie eine HTML Datei mit einer zunächst leeren Überschrift 1. Ordnung.
  • Fügen Sie mit PHP den Text '"Hallo Welt!"' inklusive der doppelten Anführungszeichen ein.
  • Versehen Sie über PHP den Text zusätzlich mit dem HTML-Markup für Betonung.
  • Speichern Sie diese Datei als PHP-Datei auf Ihrem Testserver.
  • Rufen Sie die Datei auf dem Testserver auf und kontrollieren Sie den Quelltext.
  • Rufen Sie die Datei aus dem Dateisystem heraus auf und kontrollieren Sie ebenfalls den Quelltext.

Rekapitulieren Sie die Arbeitsweise des Webservers beim Umgang mit PHP-Dateien.
