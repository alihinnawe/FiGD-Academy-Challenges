PHP-Übung - Variablen und Operatoren

Aufgabe 01

  • Legen Sie Variablen für Ihren Namen, den Kurs und die Weiterbildungseinrichtung an.
  • Nutzen Sie diese bei der Ausgaben eines Satzes in der Form: 'name besucht den Kurs kurs am/an der weiterbildungseinrichtung'

Benutzen Sie zur Ausgabe einfache Anführungszeichen.

Aufgabe 02

  • Legen Sie Variablen mit zwei Zahlen an
  • Geben Sie das Produkt dieser Zahlen in der Form '3 mal 4 ist 12' aus.

Benutzen Sie diesmal zur Ausgabe doppelte Anführungszeichen.

Aufgabe 03

  • Legen Sie eine Variable mit der Zahl 1 an.
  • Geben Sie unter Verwendung dieser Variable drei Überschriften 2. Ordnung aus:
    • 'Kapitel 1'
    • 'Kapitel 2'
    • 'Kapitel 3'

Benutzen Sie dafür den Inkrement-Operator.

Wie müssen Sie den Inkrement-Operator einsetzen wenn mit dem Startwert 0 die gleiche Ausgabe erzeugt werden soll?

Aufgabe 04

  • Legen Sie eine Variable mit der Zeichenkette 'false' an.
  • Geben Sie die Variable mit echo aus.
  • Kontrollieren Sie die Variable mit var_dump().
  • Casten Sie die Variable zu einem booleschen Datentyp.
  • Geben Sie die Variable mit echo aus.
  • Kontrollieren Sie die Variable mit var_dump().
  • Erklären Sie das Ergebnis.

Wiederholen Sie das mit einer leeren Zeichenkette.
