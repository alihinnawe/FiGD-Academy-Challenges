CSS - Übung - Tabellen

Themen

  • Tabellenformatierung

Aufgabe 1

  • Übernehmen Sie die unten angegebene HTML-Datei.
  • Fügen Sie in die Tabelle die Tabellenüberschrift 'Immobilienangebot' ein (HTML).
  • Setzen Sie diese Überschrift mittels CSS unter die Tabelle.
  • Versehen Sie Tabelle und Zellen mit einem 1px breitem Rahmen.
  • Sorgen Sie dafür, dass die Rahmenlinien zusammenfallen (überlappen).
  • Kopf- und Fußzeile sollen farbig und in komplett fetter Schrift erscheinen.
  • Versehen sie die Zellen und die Tabellenbeschriftung mit einem Innenabstand.
  • Validieren Sie.
  • Vergleichen Sie das Aussehen mit der Vorlage.
Rahmen?

Sammelangabe möglich, z.B. border:1px solid blue;

Innenabstand?

padding:Maßangabe;
