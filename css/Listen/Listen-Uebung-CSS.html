Übernehmen Sie die unten angegebenen HTML-Datei und formatieren Sie die Listen mittels CSS nach folgenden Vorgaben.

  • Die nummerierte Liste erhält eine Nummerierung mit kleinen römischen Zahlen.
  • Die in ihr verschachtelte Liste erhält eine 'Nummerierung' mit a, b ....
  • Die Aufzählungsliste erhält die unten angegebene Grafik als Aufzählungszeichen.
  • Die Aufzählungszeichen sind im Text eingerückt.
  • Validieren Sie.
  • Vergleichen Sie das Aussehen mit der Vorlage.
